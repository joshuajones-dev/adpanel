# ADPanel (Active Directory Management System)

A web-based platform for managing Active Directory users and attributes.

## Features
- LDAP integration
- User lifecycle management
- Role-based permissions
- Administrative interface

## Purpose
Built to improve operational efficiency and simplify Active Directory management through a custom web interface.

## Notes

This repository contains a simplified and sanitized version of the project. Configuration values, credentials, and environment-specific logic have been removed or replaced for security.

## Architecture Overview

- MVC-inspired application structure
- `app/` contains application logic and feature components
- `core/` contains shared framework/bootstrap functionality
- `public/` contains the web entry point and public-facing assets
